## Run Locally

Clone the project

```bash
git clone https://github.com/purpl3F0x/alanc.git
```

Go to the project directory

```bash
cd alanc-master
```

Run with cargo
```bash
cargo run alanc --release  -- <Program arguments>
```

Build with cargo
```
cargo build --release
```

__Warning: By default release builds are build for x86_64_v3 (avx,avx2) with lto__
